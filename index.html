Mutable methods

Mutable (o'zgartiriladigan) metodlar arrayni yoki obyektni bevosita o'zgartiradi.

Push () methodi

let raqamlar = [1, 2, 3, 4]; // Raqamlar ro'yxati

raqamlar. push(5) // 5 raqamini qo'shamiz

console.log( raqamlar ) // Natija: [1, 2, 3, 4, 5]

console.log( raqamlar.length ) // Natija: 5

push() metodi massivning oxiriga yangi element qo'shadi va massivning yangilangan uzunligini qaytaradi.

Pop () methodi

let raqamlar = [1, 2, 3, 4]; // Raqamlar ro'yxati

let olinganElement = raqamlar. pop() // Oxirgi elementni olib tashlaymiz

console.log( raqamlar ) // Natija: [1, 2, 3]

console.log( olinganElement ) // Natija: 4

pop() metodi massivning oxirgi elementini olib tashlaydi va shu olib tashlangan elementni qaytaradi.

Shift () methodi

let raqamlar = [1, 2, 3, 4]; // Raqamlar ro'yxati

let olinganElement = raqamlar. shift() // Birinchi elementni olib tashlaymiz

console.log( raqamlar ) // Natija: [2, 3, 4]

console.log( olinganElement ) // Natija: 1

shift() metodi massivning birinchi elementini olib tashlaydi va olib tashlangan elementni qaytaradi.

Unshift () methodi

let raqamlar = [2, 3, 4]; // Raqamlar ro'yxati

let yangiUzunlik = raqamlar. unshift(1) // Massiv boshiga 1 qo'shamiz

console.log( raqamlar ) // Natija: [1, 2, 3, 4]

console.log( yangiUzunlik ) // Natija: 4

unshift() metodi massivning boshiga yangi element(lar) qo'shadi va massivning yangilangan uzunligini qaytaradi.

Splice () methodi

let raqamlar = [1, 2, 3, 4, 5]; // Raqamlar ro'yxati

raqamlar .splice(2, 1, 6); // 2-indexdan boshlab 1 elementni o'chiradi va o'rniga 6 ni qo'shadi

console.log( raqamlar ) // [1, 2, 6, 4, 5]

splice() metodi JavaScriptda arraydan elementlarni o'chirish, qo'shish yoki almashtirish uchun ishlatiladi.

Reverse () methodi

let raqamlar = [1, 2, 3, 4, 5]; // Raqamlar ro'yxati

raqamlar .reverse(); // arrayni teskari tartibda qaytaradi

console.log( raqamlar )// [5, 4, 3, 2, 1]

reverse() metodi JavaScriptda arrayni teskari tartibda qaytarish uchun ishlatiladi.

Sort () methodi

let raqamlar = [5, 3, 8, 1, 2]; // Raqamlar ro'yxati

raqamlar .sort(); // arrayni string sifatida tartiblaydi

console.log( raqamlar )// [1, 2, 3, 5, 8]

sort() metodi JavaScriptda arrayni tartibga solish uchun ishlatiladi. Bu metod arraydagi elementlarni alifbo tartibida yoki default tarzda (ya'ni, string sifatida) tartiblaydi.

Agar raqamli arrayni tartiblayotgan bo'lsangiz, raqamlarni to'g'ri tartibda olish uchun sort() metodiga solishtiruvchi funksiyani kiritish zarur:

let raqamlar = [5, 3, 8, 1, 2]; // Raqamlar ro'yxati

raqamlar .sort( (a, b) => a - b ); // raqamlarni o'sish tartibida tartiblash

console.log( raqamlar )// [1, 2, 3, 5, 8]
