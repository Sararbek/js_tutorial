Immutable Methods

Immutable (o'zgartirilmaydigan) metodlar esa array yoki obyektni o'zgartirmaydi, faqat yangi qiymat qaytaradi

Concat () methodi

let raqamlar1 = [1, 2, 3];

let raqamlar2 = [4, 5, 6]; // Raqamlar ro'yxati

let combined= raqamlar1 .concat( raqamlar2 ) // raqamlar1 va raqamlar2 ni birlashtiradi

console.log( combined ) /// [1, 2, 3, 4, 5, 6]

console.log( raqamlar1 ) // [1, 2, 3] (o'zgarmaydi)

console.log( raqamlar2 ) // [4, 5, 6] (o'zgarmaydi)

concat() metodi JavaScriptda ikki yoki undan ko‘p arraylarni birlashtirib, yangi array yaratadi va asl arraylarni o‘zgartirmaydi.

Slice () methodi

let raqamlar = [1, 2, 3, 4, 5]; // Raqamlar ro'yxati

let sliced = raqamlar. slice( 1, 4 ) // 1-indexdan boshlab 4-indexgacha (4 o'zi kirmaydi) kesadi

console.log( sliced ) // [2, 3, 4]

console.log( raqamlar ) // [1, 2, 3, 4, 5] (asl array o'zgarmaydi)

slice() metodi JavaScriptda arrayning belgilangan qismidan yangi array yaratadi, lekin asl arrayni o'zgartirmaydi.

Map () methodi

let raqamlar = [1, 2, 3, 4, 5]; // Raqamlar ro'yxati

let squared = raqamlar. map( num => num * num ) // har bir elementni kvadratga ko'paytiradi

console.log( squared ) // [1, 4, 9, 16, 25]

console.log( raqamlar ) // [1, 2, 3, 4, 5] (asl array o'zgarmaydi)

map() metodi JavaScriptda arraydagi har bir elementga berilgan funksiya qo‘llanib, yangi array yaratadi. Asl array o'zgarmaydi.

Filter () methodi

let raqamlar = [1, 2, 3, 4, 5]; // Raqamlar ro'yxati

let evenNumbers = raqamlar. filter( num => num % 2 === 0 ) // faqat juft sonlarni tanlaydi

console.log( evenNumbers ) // [2, 4]

console.log( raqamlar ) // [1, 2, 3, 4, 5] (asl array o'zgarmaydi)

filter() metodi JavaScriptda arraydagi elementlarni berilgan shartga mos kelishiga qarab saralab, yangi array yaratadi. Asl array o'zgarmaydi.

Reduce () methodi

let raqamlar = [1, 2, 3, 4, 5]; // Raqamlar ro'yxati

let sum = raqamlar .reduce( (accumulator, current) => accumulator + current, 0 );
accumulator - yig‘ib boruvchi (boshlang‘ich qiymat bilan boshlaydi, keyin yig‘imni saqlaydi).
current - hozirgi element.
0 - boshlang‘ich qiymat (ixtiyoriy, kerak bo‘lsa ko‘rsatiladi).

console.log( sum ) / // 15

reduce() metodi JavaScriptda arraydagi elementlarni birlashtirish yoki yig‘ish uchun ishlatiladi. U berilgan funksiyani ketma-ket har bir elementga qo‘llab, bitta yakuniy qiymat hosil qiladi.

Some () methodi

let raqamlar = [1, 2, 3, 4, 5]; // Raqamlar ro'yxati

let hasEven = raqamlar .some( num => num % 2 === 0 ); // juft son bormi, tekshiradi

console.log( hasEven )// true (2 va 4 juft sonlar)

Agar arraydagi kamida bitta element shartni qanoatlantirsa, some() true qaytaradi. Aks holda, false qaytaradi.

some() metodi JavaScriptda arraydagi elementlarning kamida bittasi berilgan shartga mos keladimi, shuni tekshiradi va true yoki false qaytaradi

Every () methodi

let raqamlar = [2, 4, 6, 8]; // Raqamlar ro'yxati

let allEven = raqamlar .every( num => num % 2 === 0 ); barcha elementlar juftmi, tekshiradi

console.log( allEven )// true (hamma elementlar juft)

Agar arraydagi barcha elementlar shartni qanoatlantirsa, every() true qaytaradi. Agar birgina element shartga mos kelmasa, false qaytaradi.

every() metodi JavaScriptda arraydagi har bir element berilgan shartga mos keladimi, shuni tekshiradi va true yoki false qaytaradi

Find () methodi

let raqamlar = [1, 3, 5, 6, 7]; // Raqamlar ro'yxati

let firstEven = raqamlar .find( num => num % 2 === 0 ); // birinchi juft sonni topadi

console.log( firstEven )// 6

find() arrayni boshidan boshlab birma-bir tekshiradi.
Shartni qanoatlantirgan birinchi element topilgach, qidirishni to'xtatadi.
Agar shartga mos element bo'lmasa, undefined qaytaradi.

find() metodi JavaScriptda arraydagi birinchi elementni berilgan shartga mos kelishiga qarab topadi va o'sha elementni qaytaradi. Agar hech qaysi element mos kelmasa, undefined qaytaradi.

FindIndex () methodi

let raqamlar = [1, 3, 5, 6, 7]; // Raqamlar ro'yxati

let firstEvenIndex = raqamlar .findIndex( num => num % 2 === 0 ); // birinchi juft sonning indeksini topadi

console.log( firstEvenIndex )// 3

findIndex() arrayni boshidan boshlab birma-bir tekshiradi.
Shartga mos kelgan birinchi elementning indeksi topilgach, qidirishni to'xtatadi.
Agar shartga mos element topilmasa, -1 qaytaradi.

findIndex() metodi JavaScriptda arraydagi birinchi elementning indeksini berilgan shartga mos kelishiga qarab qaytaradi. Agar hech qaysi element shartga mos kelmasa, -1 qaytaradi

includes () methodi

let raqamlar = [1, 2, 3, 4, 5]; // Raqamlar ro'yxati

let hasThree = raqamlar .includes( 3 ); // 3 qiymati mavjudligini tekshiradi

console.log( hasThree )// true

includes(value) metodi berilgan value qiymati arrayda mavjud bo‘lsa, true qaytaradi.
Agar qiymat mavjud bo'lmasa, false qaytaradi
includes() faqat qiymatni tekshiradi indeksni emas.

includes() metodi JavaScriptda arrayda ma'lum bir qiymat mavjudligini tekshiradi va natijaga qarab, true yoki false qaytaradi

indexOf () methodi

let raqamlar = [10, 20, 30, 40, 50]; // Raqamlar ro'yxati

let index = raqamlar .indexOf( 30 ); // 30 qiymatining indeksini topadi

console.log( index )// 2

indexOf(value) metodi arrayda berilgan value qiymatini qidiradi.
Agar qiymat arrayda mavjud bo‘lsa, uning birinchi uchrashgan indeksini qaytaradi.
Agar qiymat mavjud bo‘lmasa, -1 qaytaradi

indexOf() metodi JavaScriptda arrayda berilgan qiymatning birinchi uchrashgan indeksini qaytaradi. Agar qiymat mavjud bo'lmasa, -1 qaytaradi

toString () methodi

let raqamlar = [1, 2, 3, 4, 5]; // Raqamlar ro'yxati

let str = raqamlar .toString(); // arrayni stringga aylantiradi

console.log( str )// "1,2,3,4,5"

toString() array yoki boshqa obyektlarni string ko‘rinishida ko‘rsatish uchun ishlatiladi.
Array uchun, har bir element vergul bilan ajratilgan ro‘yxat sifatida qaytariladi.
Asl array o‘zgarmaydi, faqat yangi string qiymati olinadi.

toString() metodi JavaScriptda arrayni yoki obyektni string (matn) ko‘rinishida qaytaradi. Array bo‘lsa, elementlarni vergul bilan ajratib ko‘rsatadi.

Join () methodi

let raqamlar = [1, 2, 3, 4, 5]; // Raqamlar ro'yxati

let str = raqamlar .join( '-' ); // array elementlarini "-" bilan ajratib birlashtiradi

console.log( str )// "1-2-3-4-5"

join() arraydagi elementlarni birlashtirib, belgilangan separator yordamida string hosil qiladi.
Agar separatorni ko‘rsatmasangiz, default qiymat sifatida vergul (,)ishlatiladi

Misol: Separator ko‘rsatilmasdan:

let raqamlar = [1, 2, 3, 4, 5]; // Raqamlar ro'yxati

let str = raqamlar .join(); // vergul bilan ajratilgan string

console.log( str )// "1,2,3,4,5"

join() arraydagi elementlarni birlashtirib, belgilangan separator yordamida string hosil qiladi.
Agar separatorni ko‘rsatmasangiz, default qiymat sifatida vergul (,)ishlatiladi

join() metodi JavaScriptda arraydagi elementlarni birlashtirib, ularni biror ajratuvchi (separator) bilan ulaydi va yangi string qaytaradi. Agar ajratuvchi ko‘rsatilmasa, elementlar vergul bilan ajratiladi.
